# Vision produit — MAESTRO

## Problème

Les agences de marketing gérant des clients HORECA produisent du contenu social media manuellement, de façon répétitive, pour chaque restaurant, hôtel ou bar. C'est lent, coûteux et incohérent avec l'identité de chaque marque.

## Solution

MAESTRO automatise la production de contenus via des agents IA spécialisés, tout en gardant l'humain au centre des décisions de publication.

## Positionnement

- **Cible V1** : agences de communication gérant 3 à 30 clients HORECA. Outil interne, un utilisateur admin.
- **Cible V2** : plateforme SaaS multi-utilisateurs avec portail client, facturation et white-label.

## Valeur différenciante

1. **Direction Artistique (DA) par client** — MAESTRO apprend le style visuel de chaque client à partir de ses photos réelles. Chaque image générée respecte cette DA.
2. **Learning loop** — Les performances des posts publiés alimentent les prochaines générations. Ce qui a bien marché est réutilisé.
3. **Supervisor IA** — Chaque post passe par un agent de contrôle qualité avant validation humaine. L'agence ne publie jamais un contenu hors-charte sans le voir.
4. **Pipeline traçable** — Chaque étape de génération est loggée : agent, durée, coût, verdict.

## Workflow cible

```
Profil client
→ Stratégie + ADN de marque
→ Library (photos + vidéos + DA)
→ Studio (brief → génération → preview)
→ Validation (supervisor IA + validation humaine)
→ Calendrier (programmation)
→ Publication Meta (Facebook + Instagram)
→ Analytics + apprentissage
```

## Ce que MAESTRO n'est pas (encore)

- Pas un outil de gestion de communauté (pas de réponse aux commentaires).
- Pas un outil de veille ou de curation.
- Pas un outil multicanal complet (TikTok, LinkedIn en future roadmap).
- Pas une plateforme SaaS en V1.
